# riscv-tests

Prebuilt RISC-V test binaries for RV32I instruction tests.

This repository is based on [riscv-software-src/riscv-tests](https://github.com/riscv-software-src/riscv-tests).

The generated `.hex` files are intended to be loaded into simulator memory and executed by a custom RISC-V CPU implementation.

Passing these tests does not guarantee that the CPU implementation is completely correct.
